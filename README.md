## Installation & Setup
If you haven't installed gulpjs first, you can install it [here](https://gulpjs.com/docs/en/getting-started/quick-start).
```bash
git clone https://github.com/ziarparvaiz/github-dev.git
cd github-dev
npm install
gulp
```
